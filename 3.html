JavaScipt

javascript는 html을 제어하는 언어이다.

input이란 태그를 사용해 다음 코드들을 보일 수 있다.
input 다음에는 type를 사용하여 속성을 부여할 수 있다.
type 다음에는 value를 사용해 버튼의 이름을 정할 수 있다.
value 다음에는 onclick은 웹 브라우져의 html사용 설명서에는 onclick이란 속성의 값으로는 자바스크립트가 와야된다고 설명되있다.
onclick 이라고 하는 속성의 값으로
document.querySelector('body').style.backgroundColor='black';
document.querySelector('body').style.color='white';
라는 자바스크립트를 넣어주면 사용자가 onclick속성이 위치한 버튼을 클릭했을때 위의 자바스크립트를 실행시킨다.
document.querySelector('body').style.backgroundColor='black';라는 자바스크립트 코드를 해석 해보면
document(이 문서)에서body(바디 태그)를 Select(선택)하는 코드가document.querySelector('body')이다.
(bady는 말 그대로 <bady>이다.)
(이 바디태그에)style(속성 값)으로 backgroundcolor을 black으로 한다.
자바스크립트 코드는 알파벳의 대소문자 비교한다.

기본적으로 자바스크립트는 html위에서 동작하는 언어로 시작되었다.
html에 완전히 다른 문법인 자바스크립트를 사용하기 위해서는 웹 브라우저에게 html코드로 자바스크립트가 시작된다는 것을 알려야 된다.
자바스크립트를 사용할때에는 <script> </script> 사이에 자바스크립트 언어를 입력한다.
자바스크립트의 장점은계산기 등의 다양한 기능이 가능하다.
[EX:기본적으로 html의 1+1을 입력할 시 1+1이 나오지만 자바스크립트의 1+1 은 2가 나온다.]

자바스크립트에서 어떤 글씨를 출력할 때에는 document.write(abc);를 쓴다.(<>는 사용하지 않는다.)
